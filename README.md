## Development

```bash
# install
pnpm install

# dev -- use tsx compiler .ts file in command line
pnpm dev

# build -- use rollup build
pnpm build

# test -- use vitest
pnpm test

# format -- use prettier
pnpm format

# dev:cjs -- example
# "type": "module",  // if add type: module; dev:cjs can't work
```
